# Arch Linux

## Yay
```sh
$ yay -R <package name>   # remove a package
$ yay -Rc <package name>  # Remove package and all packages that depend on it
$ yay -Sc                 # clean all old packages
$ yay -Qi <package name>  # check if package is installed
$ yay -Qi                 # list all installed packages
$ yay -S                  # install OR update package
$ yay -Ss                 # search for package
```
